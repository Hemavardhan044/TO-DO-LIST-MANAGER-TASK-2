# To-Do List Application

## Description

This is a simple console-based To-Do List application written in Python. It helps users manage tasks by allowing them to add, view, and remove tasks. Tasks are saved in a text file so they remain available after the program is closed.

## Features

* View tasks
* Add new tasks
* Remove tasks
* Save tasks automatically
* Load saved tasks when the program starts

## Files

* `todo.py` - Main Python program
* `tasks.txt` - Stores tasks permanently

## Requirements

* Python 3.x

## How to Run

Open a terminal in the project folder and run:

```bash
python todo.py
```

## Menu Options

1. View tasks
2. Add task
3. Remove task
4. Exit

## Example

```text
--- To-Do List ---
1. View tasks
2. Add task
3. Remove task
4. Exit

Choose (1-4): 2
Enter task: Complete assignment
Task #1 added: Complete assignment
```

## Outcome

A simple persistent To-Do List application that stores tasks in a text file and allows basic task management through the command line.
